Question - In a school, students decided to plant trees in and around the school to reduce air pollution. It was decided that the number of trees, that each section of each class will plant, will be double of the class in which they are studying. If there are 1to 12 classes in the school and each class has two sections, find how many trees were planted by the students. Which value is shown in this question?
